Preformatted text

Preformatted text keeps the format in which text appears in HTML and does not attempt to reformat it. For example, this is a poem inside a paragraph that is supposed to have whitespace between it..... maxling it is with a heart heavy that i admit loss of a feline so loved a friend lost to the unknown (night) ~cdr 11dec07

Blockquote

A block quote is simply a quote from another author. It will indent the quote. Here is an example

                        "That monster custom, who all sense doth eat
                        Of habit's devil,"

                        "What a falling off was there !
                        From me, whose love was of that dignity
                        That it went hand in hand even with the vow
                        I made to her in marriage, and to decline
                        Upon a wretch."

Ordered List

An ordered list lists items in order. Here is an example

My team members are

  1. Steven
  2. Patrick
  3. Jesus
  4. Alvin
  5. Gevorg

Unordered List

An unordered list lists items in any order. Here is an example

My team members are

  • Patrick
  • Jesus
  • Alvin
  • Steven
  • Gevorg

List Items

A list item goes inside either an ordered or an unordered list. It is the list item inside the list

  • This is list item 1
  • This is list item 2
  • This is list item 3

Description List

A description list respresents zero or more name-value groups such as a description term or a description definition

Authors
John
Luke
Editor

Description Term

The description term is the name in a description group. The lines that ask a question are the Description terms

What do we want?
Our data.
When do we want it?
Now.
Where is it?
We are not sure.

Description Definition

The Description definition is the value that goes with the name, or the description term.

Authors
Gevorg

Editors

Div

Main

The main element represents the main content of a body. It cannot be presented visually but it is used in this document.
